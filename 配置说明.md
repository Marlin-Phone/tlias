# 配置说明

该项目为**前后端分离**的项目，包含前端和后端两部分代码，分别位于 `frontend` 和 `backend` 目录下。  
启动该项目前需分别进行**前端配置**和**后端配置**及**数据库初始化**。

## frontend（前端）

前端代码位于 `frontend` 目录下，使用 Vue 框架开发。主要负责用户界面和交互逻辑。

### 运行前的准备工作

确认安装 Node.js 和 npm。可以通过以下命令检查是否已安装：

```bash
node -v
npm -v
```

如果未安装，请前往 [Node.js 官网](https://nodejs.org/) 下载并安装最新版本。
确认安装后则可进入下一步。

## backend（后端）

后端代码位于 `backend` 目录下，使用 Node.js 和 Express 框架开发。主要负责处理业务逻辑和数据存储。

### 运行前的准备工作

- 确认 [application.xml](backend/src/main/resources/application.yml) 中的数据库配置正确(url、username、password)，确保能够连接到数据库。
- 确认OSS的配置正确(endpoint、bucketName、region)，确保能够连接到阿里云OSS服务。

以下配置信息较为简略，但都来自于官方文档。详细信息请参考阿里云OSS官方文档进行配置。

官方文档：https://help.aliyun.com/zh/oss/developer-reference/oss-sdk-for-java-2-0/

需要在**本地配置**阿里云OSS的key和secret。

1. 首先去阿里云申请到Access Key ID和Access Key Secret。

2. 在CMD中运行以下命令。

```cmd
setx OSS_ACCESS_KEY_ID "YOUR_ACCESS_KEY_ID"
setx OSS_ACCESS_KEY_SECRET "YOUR_ACCESS_KEY_SECRET"
```
3. 运行以下命令，检查环境变量是否生效。

```cmd
echo %OSS_ACCESS_KEY_ID%
echo %OSS_ACCESS_KEY_SECRET%
```
## 数据库初始化

请确保已安装 MySQL 8.0，并创建名为 `tlias` 的数据库。  
数据库脚本位于[init.sql](init.sql)下，运行该sql脚本即可创建所需要的表并添加初始数据。
